# DisposAI - AI-Powered Medicine Disposal Guide

An intelligent application that helps users safely dispose of medications using AI-powered identification and guidance.

## Project info

**URL**: https://lovable.dev/projects/REPLACE_WITH_PROJECT_ID

## üöÄ AI-Powered Features

This application uses **OpenAI GPT** and **Tesseract.js OCR** to provide:

- **Real-time OCR**: Extracts text from medicine package images
- **AI Medicine Identification**: GPT identifies medicines from detected text
- **Intelligent Disposal Guidance**: AI-generated disposal methods, warnings, and environmental impact data
- **Location-Based Pharmacy Finder**: GPT-powered search for nearby take-back locations
- **Dynamic Information**: All medicine data is generated by AI, not hardcoded

### 1. Install Dependencies

```sh

```

### 2. Configure OpenAI API Key

Create a `.env` file in the root directory:

```sh
# .env
VITE_OPENAI_API_KEY=your_openai_api_key_here
```

**Get your API key:**
1. Visit https://platform.openai.com/api-keys
2. Sign up or log in
3. Create a new API key
4. Copy it to your `.env` file

**Important:** The API key must start with `VITE_` prefix to be accessible in Vite applications.

### 3. Start Development Server

```sh
npm run dev
```
